# PRD 요약: 서울형 통합 돌봄 중개 플랫폼 (서울케어플러스)

## 비전
"서울시 공공 데이터의 초정밀성과 지역 주민의 검증된 경험을 연결하여, 내 집 앞 가장 안전한 돌봄을 찾는다."

## 단계별 전략
- **Phase 1 (MVP)**: 서울시 전용 서비스
- **Phase 2 (Scale-up)**: 경기도 확산(경기데이터드림 연동)
- **Phase 3 (National)**: 전국 확장(공공데이터포털/빅데이터 플랫폼)

## 핵심 사용자군
- 학부모(아동 돌봄)
- 보호자/가족(노인 돌봄)
- 지역 주민(복지/문화 서비스 이용자)

## MVP 기능 범위

### 1) 아동 돌봄
- 서울시 어린이집 상세 속성 제공
- 스쿨존 CCTV 시각화
- 사고 다발 구간 회피 경로 제공

### 2) 노인 돌봄
- 요양시설 인증/공실 상태 제공
- AED/응급실 접근성 노출
- 골든타임 안심 지수 제공

### 3) 지역사회
- 공공서비스 예약 정보 연동
- 무장애 동선/접근성 정보 연동

### 4) 신뢰·안전
- 행정동 기반 주민 인증 리뷰
- 050 안심번호
- robots.txt 준수 외부 데이터 수집

### 5) 수익화
- 동 단위 로컬 광고
- 프리미엄 리포트 구독
- O2O 긴급 간병인 매칭 수수료
