# MVP 실행 계획 (3개월)

## 1개월차: 데이터 댐 구축
- 서울 열린데이터 API 키/접근정책 정리
- 핵심 데이터셋(어린이집/요양시설/안전인프라) 수집 파이프라인 구축
- 목표: 시설 5,000개소 적재 + 일배치 안정화

### 완료 기준
- 배치 성공률 95%+
- 필수 컬럼 결측률 3% 미만
- 시설 데이터 지도 시각화 가능

## 2개월차: 플랫폼 개발
- 시설 검색/상세/비교 API
- 안심 통학 지도 + 골든타임 지수 API
- 리뷰 작성/검증(주민 인증+OCR) + 050 안심번호 연동

### 완료 기준
- 검색 응답 p95 < 500ms (캐시 히트 시)
- 리뷰 검증 파이프라인 SLA 5분 이내
- 핵심 사용자 플로우 E2E 테스트 통과

## 3개월차: 베타 오픈/검증
- 타겟 구(예: 노원/송파) 한정 공개
- 맘카페/시니어 커뮤니티 제휴 이벤트 운영
- KPI 수집 및 모델/정책 보정

### KPI
- 초기 진성 리뷰 1,000건
- 재방문율(주간) 25%+
- 문의→매칭 전환율 10%+

## 리스크 및 대응
- 데이터 스키마 변경: adapter 버전 관리/자동 계약 테스트
- 리뷰 악성행위: 가중치 기반 신뢰점수 + 운영자 승인 게이트
- 응답지연: 캐시 계층/비동기 사전계산 확장
